# Factory Firmware for reTerminal D1001
```
This example based on the development of esp_brookesia_phone. 
This factory firmware demonstrates the peripheral functions of the device. 
Some peripheral working need micro sd card to store file. 
And the device will format sd card to FAT32, back up your data first.
```

## Features
```
- LCD display and touch response
- Camera viedo and screenshot
- Microphone record and playback
- WiFi connect to the network
- LTE PPP connect to the network
- Accelerometer and gyroscope raw data
- Battery management
- Local real rtc
```
